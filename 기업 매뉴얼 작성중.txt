1. 프로젝트 등록
  프로젝트 등록은 어떻게 하나요?
    - 프로젝트 등록 관련 폼이 나온 이후 추가 등록
    예시
    - 개요(기본정보) : 이미지, 제목, 카테고리, 모집인원, 등?
    - 스토리 작성 : 프로젝트 영상, 핵심 메시지, 스토리
    (이건 각 페이지 별 메시지 들을 받아야겠다)
    - 참여 조건 : 기본조건(성별,나이,지역,직업 등), 필수확인 조건 입력
    - 피드백 받을 카테고리? 선택
    - 링크 및 테스트 내용
    - 미리보기
    - 결제
  프로젝트 가격은 어떻게 책정되나요?
    - 가격정책을 공개 한 뒤
    - 프로젝트 메뉴 => 가격명세서 버튼을 통해 확인 설명
?  프로젝트 수정은 어떻게 하나요?
    - 등록 완료 이전 / 등록 완료 이후 설명
    - 등록 완료 이전에는 수정 하려면 어떤 과정을 거쳐야 하는지
?  프로젝트 결제 및 환불은 어떻게 하나요?
    - 결제 방법 안내
    - 환불 방법 안내

2. 프로젝트 확인
  등록 완료된 프로젝트는 어떻게 확인하나요?
    - 프로젝트 버튼을 통해 기업이 작성한 프로젝트들의 목록에서 이동
    - 마이페이지에서 이동
  작성한 스토리는 어떻게 확인하나요?
    - 프로젝트 화면에서 스토리보기 버튼
    - 스토리 보기 기능들의 예시를 통해 보여주자
  참여 신청한 사람들의 정보는 어떻게 확인하나요?
    - 프로젝트의 메뉴에서 각종 정보를 확인할 수 있는 방법 안내

3. 피드백 및 인터뷰 (=프로젝트 진행)
  참여한 사람들의 피드백은 어떻게 확인하나요?
    - 프로젝트 메인에서 스크롤 내리면 볼 수 있는 것
    - 토론에서 공감을 많이 받은 피드백은 best로 보이는 것 안내
    - 우측 상단 알림에서 바로가기 가능한 점 안내(이건 근데 프로젝트 홈으로 가네?)
  원하는 피드백을 따로 볼 수 있나요?
    - 프로젝트 검색 버튼을 통하여 각 카테고리 별 선택해서 볼 수 있는 것 안내
  참여자들의 토론은 어떻게 확인하나요?
    - 각 피드백의 상세보기 버튼을 통해 피드백으로 들어가서
    - 공감 비공감에 관하여 어떻게 확인할 수 있는지 안내
? 이상하게 작성한 피드백에 관하여??

  인터뷰는 어떻게 진행되나요?
    - 기업 <-> 유저 간 대화 형식이라는 간단한 프로세스 이미지
    - 인터뷰를 해야하는 이유 및 장점 설명
    - 인터뷰에 들어가면 좋은 내용이나 자세들에 대한 안내(링크도 좋을 듯)
  인터뷰는 어떻게 신청하나요?
    - 프로젝트 홈의 우측 하단에서 신청하는 방법
    - 피드백을 남긴 유저의 이름을 눌러서 들어가서 신청하는 방법
    - 메인에서 인터뷰 탭을 통해 프로젝트 선택, 유저 선택 후 인터뷰 하는 방법
    - 여기도 들어가야 할 내용 예시 안내해주면 좋겠네
  인터뷰는 어떻게 확인하나요?
    - 새로운 응답이 왔으면 인터뷰 쪽에 새로운 알림이 있다는 표시를 보여주고
    - 인터뷰 확인하러 가는 과정 안내
    - 우측 상단 알림에서 바로 가기
? 인터뷰는 얼만큼 신청할 수 있나요?
    - 요건 어뜨켕?
    - 프로젝트 별 인터뷰 부분에서 우측 상단에 몇 개중 몇 개 이용했다는것 알려줘

4. 프로젝트 결과 확인
  프로젝트 결과물은 어떻게 확인하나요?
    - 메뉴에서 결과보고서 확인 하는 방법
    - 결과 보고서에 들어가는 내용들 안내
  각각의 정보를 따로 확인 가능하나요?
